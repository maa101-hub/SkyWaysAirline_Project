# SkyWaysAirline_Project

A comprehensive airline management system project featuring a React + Vite frontend.

## 📋 Project Overview

SkyWaysAirline is a web-based airline management and booking system with a modern, responsive user interface.

## 🚀 Getting Started

### Frontend Setup (React + Vite)

The frontend is built with **React** and **Vite** for optimal development experience.

#### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

#### Installation
```bash
cd SkyWaysLine-frontend
npm install
npm run dev
```

## 📦 Tech Stack

- **Frontend**: React + Vite
- **Build Tool**: Vite with HMR (Hot Module Replacement)
- **Linting**: ESLint configured

## 🔧 Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
